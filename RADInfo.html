Radikalfun



Este proyecto consistió en el rediseño del branding para la marca de ropa y la tienda de productos para motos Radikalfun.


Para la realización del logotipo realizamos un trabajo de lettering a partir de la base de una tipografía ya existente, la Avant Garde. Para ello estuve realizando diversas pruebas hasta que finalmente a partir de la Avant Garde Italic encontré unas formas dinámicas y angulosas para poder representar la verdadera esencia de la marca, la cual fue concebida para satisfacer las necesidades de los deportistas que realizan deportes extremos y radicales.


A partir de este lettering realizamos la versión vertical del logotipo que es para la tienda y la versión RAD la cual tiene una serie de modificaciones ópticas para que se vea más equilibrado y solamente se utiliza para la ropa, este va acompañado de un símbolo que nace a partir de la forma de la "K" y representa el logotipo anterior de la marca haciendo alusión al "rad" que en la jerga americana es un gesto que se realiza con la mano, cerrando el puño y extendiendo el dedo pulgar i el meñique.


Respecto al color decidimos utilizar el blanco y el negro como colores principales para diferenciarnos de la competencia y buscar el contraste a nivel visual que también podemos conseguir con el color secundario que se utiliza en algunas aplicaciones que es el amarillo.
